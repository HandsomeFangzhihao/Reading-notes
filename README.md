# 笔记-电影/名言/阅读

>我想让每一次观看/阅读都变得意义非凡

	这个世界上，能让我颤栗的事物大体有六种—
	人性深处的黑暗与残酷，熊熊燃烧的火焰与鲜血，
	诡谲精巧的设计与布局，照射心灵的希望与救赎，
	纯净无暇的生活与爱恋，还有数不清的好吃的小鱼干。
								《布拉德·林大喵》
